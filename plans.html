Planned Curriculum

Here's an excerpt of what courses FreeGram will be offering. This is a work in progress and may change over time.

Python

FreeGram will offer a handful of Python courses aimed at helping beginners get their foot in the door in programming. Python will also be the first language covered. Some of the classes will include:

  • Introduction to Python: Basics, syntax, and simple programs.
  • Data Structures in Python: Lists, dictionaries, sets, and tuples.
  • Object-Oriented Programming: Classes, objects, inheritance, and polymorphism.
  • Web Development with Flask: Building simple web applications.
  • Data Analysis with Pandas: Handling and analyzing data sets.
