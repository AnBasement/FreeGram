Planned Curriculum

Various courses will be offered through FreeGram, with the goal of at one point being able to provide certifications. The initial plans are to provide courses in Python, HTML and CSS, as those are the main languages I'm working with while creating FreeGram, with JavaScript probably being next after that. Below you will find more details on what exactly is planned for each language. Keep in mind that everything you read here is subject to change.

Python

FreeGram will offer a handful of Python courses aimed at helping beginners get their foot in the door in programming. Python will also be the first language covered. Planned beginner courses include:

  • Python Basics: Learn about variables, data types, and simple input/output.
  • Control Flow: Introduction to if-statements, loops, and basic logic.
  • Functions: How to write and use functions in Python.
  • Working with Lists: Storing and processing collections of data.

HTML

It's useful to know at least the basics of HTML markup, which is the foundation of all web pages. Planned beginner courses include:

  • HTML Fundamentals: Structure a web page with headings, paragraphs, and lists.
  • Links and Images: Add navigation and visuals to your pages.
  • Forms (Intro): Basic user input with forms and simple controls.

CSS

CSS is what (can) make web pages look good, and it's something I've struggled with myself. Planned beginner courses include:

  • CSS Basics: Selectors, properties, and how to style text and backgrounds.
  • Box Model & Layout: Margins, padding, borders, and simple layouts.
  • Colors & Fonts: Make your site visually appealing with color and typography.
